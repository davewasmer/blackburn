blackburn (options)

Create a blackburn middleware function that will patch res.render to use the blackburn serializers and renderers.

Arguments

  • options (Object):
  • options.adapter (String,Object):
    • the adapter to use to interrogate model instances
  • options.serializers (String,Object):
    • either the path to the directory of serializers, or an object whose keys are model types, and whose values are instances of their respective Serializers. Defaults to 'serializers'. The "application" serializer will be used when no type specific serializer is found.
  • options.renderMethodName (String):
    • the name of the render method to attach to the response object. Defaults to 'render', which will override the Express default render method.

Returns

(Function)

blackburn middleware function
